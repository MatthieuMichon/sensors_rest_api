# Requirements

* Tested on Debian distro.
* Python 3
* Virtualenv

# Installation

This repo relies on Flask, installed through virtualenv with the following procedure:
```
$ cd sources
$ virtualenv --python=python3.4 flask
$ . flask/bin/activate
$ flask/bin/pip install flask
```
